## 🚀 The Project
We are building a 5-axis, artificial intelligence-powered industrial robotic arm. Moving beyond traditional joystick controls, our system utilizes a custom micro-architecture to translate human speech and visual cues into precise physical movements.

## ⚙️ The Tech Stack
Our robotic system is divided into two primary environments: the Micro-Architecture (Nerve Center) and the Backend (Brain).

* **Software & AI:** Python Flask, OpenCV, MediaPipe, OpenAI API
* **Frontend Dashboard:** Next.js, React, Tailwind CSS
* **Micro-Architecture:** ESP32 Wi-Fi Module
* **Motor Control:** PCA9685 16-Channel 12-bit PWM Driver (via I2C Protocol)
* **Hardware:** 3D-printed chassis, 3x MG995 Heavy Servos, 2x MG90S Precision Servos, 5V/3A Buck Converter
